# Simple Interest Calculator

This project provides a small Flask backend with a simple HTML/CSS/JS frontend to calculate simple interest.

Setup

1. Create a virtual environment (recommended) and activate it.

Windows (PowerShell):
```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python simple.py
```

Usage

- Open http://127.0.0.1:5000 in your browser.
- Fill in Principal, Rate (%) and Time (years) and click Calculate.
